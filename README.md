\#  Ecommerce-Automation-Framework



\##  Overview

AutoQA-Pro is a Selenium-Java Automation Framework designed for advanced UI element handling. It uses industry best practices like Page Object Model (POM), TestNG, and reusable utilities to build scalable and maintainable automation tests.



\---



\##  Tech Stack

\- Java

\- Selenium WebDriver

\- TestNG

\- Maven

\- Extent Reports / Allure (if used)

\---



\## ▶️ How to Run



1\. Clone the repository

2\. Import as Maven Project in Eclipse

3\. Run `testng.xml`
